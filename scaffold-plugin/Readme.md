# AlternateCMS Scalfold Plugin

This plugin is not designed to provide any functionality by default. It was designed to assist with building plugins for AlternateCMS.

## Steps to create a plugin for AlternateCMS

1. Download this plugin 
2. Update the index.js file with your own javascript.
3. Update the package.json file with your plugin name (to target AlternateCMS, we recommend something like @alternatecms/plugin-name), description to describe the plugin, license, and other npm details

4. Once completed, publish to npmjs.com with "npm publish --access public"

To create npm plugins follow these steps from npmjs.com: https://docs.npmjs.com/creating-node-js-modules
